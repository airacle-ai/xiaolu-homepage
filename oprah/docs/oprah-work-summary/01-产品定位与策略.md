# 产品定位与策略

## 当前形态（2026-05-23 终版）

- **产品名：** Oprah（曾用名：晴天 SunnySide）
- **一句话定位：** AI 对话式人格探测 + 关系碰撞工具，暗黑赛博·低饱和视觉风格
- **核心价值：** 用 AI 对话制造被看见感，用结构化维度建立可信度，用关系碰撞完成传播

### 五层价值模型

| 层 | 核心价值 | 关键设计 |
|----|---------|---------|
| 第一层：被理解感 | AI 对话，让用户产生"它懂我"的感觉 | 回声机制、矛盾捕捉三层、去模板化追问 |
| 第二层：可信度 | 结构化维度 + 三级证据 | 证据摘要 → 引用来源 → 反例检查 |
| 第三层：传播 | 关系碰撞 + 身份标签 | 关系名片 Canvas 导出 1080×1920 |
| 第四层：行动 | 每个洞察配微小行动 | "轻轻试试"行动建议 + 反馈按钮 |
| 第五层：留存 | 时间轴 + 变化记录 | 跨关系模式发现、反思触达 |

### 双模式架构

| | 随性漫游（原自由散步） | 深层剖视（原描绘晴天） |
|------|---------|---------|
| 时长 | 3 分钟 | 10 分钟 |
| 轮数 | 7-8 轮 | 10-15 轮 |
| 维度 | 12 核心维 | 19 维全覆盖 |
| 思考框架 | 无 | 苏格拉底诘问 + 第一性原理 + 布鲁姆金字塔 + 水平思考法 |
| 视觉 | 冰蓝描边 | 雾粉描边 |

## 演化路径

```
v1 原始 Demo（airacle-ai/oprah-demo）
  → 单一对话流程，15 道情境题硬编码在 COACH_SYSTEM_PROMPT
  → 4 位暗号，无身份标签，无证据层
  → 浅色主题，基础 UI

v2 晴天 SunnySide（产品策略扩展）
  → 五层价值模型提出
  → 身份标签 + 结构化证据 + "我不认同"出口
  → 冷启动三种碰撞模式（双人/推测/原型12角色）
  → 光遇疗愈草原 + 晴天浅白毛玻璃视觉

v3 Oprah 终版（当前）
  → 双模式入口分流
  → 去模板化对话（不再按顺序出题）
  → 人格标签热梗版（2-4字超短传播型）
  → 深层剖视模式注入四重思维框架
  → 暗黑赛博·低饱和视觉（#12141A 深炭黑 + #6376F0 冰蓝微光）
  → Supabase 数据层 + TokenRouter 真实 LLM
```
