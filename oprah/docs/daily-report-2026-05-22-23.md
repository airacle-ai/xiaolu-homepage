# Oprah 工作日报（2026-05-22 ~ 2026-05-23）

## 一、产品定位与命名演化

### 1.1 定位升级
- 原始定位：AI 对话式人格测试 demo → 五层价值模型：被理解感 → 可信度 → 传播 → 行动 → 留存
- 命名变迁：Oprah → 晴天 SunnySide → Oprah（终版回归）
- 模式命名：自由散步/描绘晴天 → 随性漫游/深层剖视
- 视觉定位：光遇疗愈风 → 暗黑赛博·低饱和潮流风

### 1.2 人格标签体系热梗化
全部人格标签从学术风格压缩为 2-4 字传播型：

| 类别 | 数量 | 示例 |
|------|------|------|
| 关系类型 | 10 个 | 世另我、天生绝配、越处越香、安心搭子、越吵越铁 |
| 身份标签 | 15 个 | 和事佬、守界人、柔叛逆、架构师、深挖人、捕手 |
| 碰撞角色 | 28 个 | 磨脑搭子、开格局、读心师、避难所、兜底人、照妖镜 |

## 二、AI 对话系统演进

### 2.1 Prompt 重写
- **去模板化：** 从"按顺序出 15 道情境题"改为"从用户当前表达出发自然追问"
- **五幕式结构 → 自由流动：** 不再硬性规定轮次和洞察密度，由对话自然节奏驱动
- **新老用户识别：** 老用户回来不再重复开场，像老朋友一样打招呼并提及身份标签
- **深层剖视模式注入四重思维框架：** 苏格拉底诘问 / 第一性原理 / 布鲁姆认知金字塔 / 水平思考法。Oprah 内化使用但不对用户暴露术语

### 2.2 对话流程
- 双模式入口：登录后选「随性漫游」（3 分钟/7-8 轮）或「深层剖视」（10 分钟/10-15 轮）
- 收束逻辑：12 维强信号触发收束，19 维全触及主动问"看结果还是继续"
- 进度条改版：数字 → 四阶段文案（轮廓初现/渐次成形/接近完整/全维已触及）
- 随时查看结果：≥6 维覆盖后进度条展开面板出现"查看结果"按钮

### 2.3 题库结构化
- 将 COACH_SYSTEM_PROMPT 中的 15 道情境题拆出为 `questionBank.ts`
- 每道题标注 `core`（8 道，简洁版用）或 `extended`（7 道，深度版用）
- 选题函数：按未覆盖维度数量自动排优先级的 `selectNextQuestion()`

## 六、不足与待改进

### 6.1 UI 设计能力薄弱（本周重点）
**问题：** 当前 UI 设计简陋，缺乏高级感和辨识度。两个版本（暗黑赛博/光遇草原）均停留在纯 CSS 层面，没有使用图片素材、纹理、插画等提升视觉层次的方法。
**方向：**
- 学习 AI 辅助设计工具 + 高质量素材库的组合使用
- 研究 Notion、Linear、Arc Browser 等现代产品设计语言
- 进阶目标：大模型在对话中逐步理解用户审美偏好，**自适应生成**匹配用户风格的形象图和对话界面风格

### 6.2 进入页缺乏吸引力
**问题：** 登录 PIN 页只有标题和输入框，无法在 3 秒内激发新用户好奇心。
**方向：** 加入动态视觉钩子（如粒子流、数据流扫描动画、线条拼出人格轮廓）、一句锋利文案制造悬念。

### 6.3 社交传播符号体系未建立
**问题：** 虽然有 10 个关系类型和 15 个身份标签，但仍是封闭体系内的分类。没有与用户已有的认知框架（星座、MBTI、八字、玛雅历等）建立映射。
**方向：** 将 19 维人格数据**转化为高传播力的社交符号**——例如"你的人格星盘"、"你的关系能量值"、"和 MBTI 的互译对照表"。让用户在已有的文化语境中找到自己的位置，从而产生传播冲动。

### 6.4 情感依附不足
**问题：** 用户做完测试、碰撞几个朋友后，缺乏持续回来的驱动力。当前留存层（时间轴/反思触达）仍是功能层面的设计，未触及情感层面。
**方向：** 创造用户"不想离开"的体验——AI 记住用户的重要时刻、情绪曲线、里程碑变化，像一本可以被对话翻阅的成长日记。

### 6.5 交互形式单一
**问题：** 当前只有文字输入一种交互方式。
**方向：** 建立语音 → 视频的多模态路线图。短期先支持语音输入（ASR），中期加入 AI 生成视频/形象回应。

### 6.6 对话体验仍有机械感
**问题：**
- 回答长度雷同，缺乏长短不一的自然节奏
- 部分追问仍有模板痕迹
- 老用户回来没有足够差异化的体验

**方向：**
- 回答长度由内容驱动，有时一句话的洞察比一段追问更有力
- 进一步降低"出题感"，增加"随意聊"的自由度
- 老用户可根据历史画像跳转到更高阶的话题

### 6.7 心理学基础需要系统化
**问题：** 当前 COACH_SYSTEM_PROMPT 有心理学素养的表述，但缺乏结构化的心理学知识层。
**方向：** 在 prompt 中嵌入依恋理论、认知行为、正念、存在主义心理学等核心框架，让 Oprah 在陪伴治愈的同时具备真正的心理学素养。

### 6.8 听觉维度缺失
**问题：** 没有声音，限制了沉浸感。
**方向：** 加入可选的 BGM 功能（提示 + 关闭按钮）。根据用户在对话中暴露的人格特质推荐匹配的古典乐：莫扎特（秩序/理性型）、贝多芬（激情/行动型）、巴赫（沉思/内省型）、德彪西（感性/直觉型）。

### 6.9 缺少轻量日常交互场景
**问题：** 并非每个人每天都有时间进行 10 轮深度对话。缺少"一句话交互"场景。
**方向：** 新增**心情日记**功能：
- 用户每天可花 30 秒说一句话、记录心情
- 支持拍照、传图、记录完成了什么
- AI 自动归纳心情曲线、周报、月度情绪总结
- 成为用户的"秘密日记"而非"聊天机器人"

### 6.10 缺少社会议题的连接能力
**问题：** 产品不具备对流行社会议题的回应能力。
**方向：** 追踪网络流行心理学/焦虑话题（如 ADHD、高敏感、讨好型人格、内耗、FOMO 等），在适当时机提供温和的专业视角。不蹭热点，而是**帮用户消化热点带来的焦虑**。
