## 一、用户需求分析

### 1.1 核心用户画像

亲子家庭财务规划的核心需求随孩子成长阶段呈现明显的阶段性分化。我们将用户划分为三个生命周期段，每个阶段的财务决策驱动力、痛点和信息需求存在本质差异。

---

#### 阶段一：0-3岁（新生期家庭）

**用户特征：**
- 主要决策者：25-35岁，双职工城市家庭，月收入1.5万-4万元
- 家庭结构：第一胎居多，部分有老人协助照料
- 信息获取：高度依赖母婴社群（小红书、宝妈群），容易被KOL内容影响
- 财务状态：刚购房或租房压力大，月储蓄率偏低（10%-20%），流动性紧张

**核心财务决策场景：**
1. **月子中心 vs 居家坐月子**：费用差异2-5万元，决策周期短，情绪压力大，缺乏ROI分析框架
2. **辅食品牌选择**：国产vs进口差价明显，无科学标准参考，常被营销内容主导
3. **早教课程报名**：0-3岁早教课单次200-500元，年费数千至2万，家长焦虑感最强的消费决策
4. **教育金何时开始储备**：多数家庭未建立教育金账户，缺乏具体规划入口
5. **保险配置**：儿童重疾险、医疗险优先级混乱，面对保险销售话术无从判断

**核心痛点：**
- 无法区分「必要支出」与「焦虑驱动支出」
- 对早教ROI没有参照系，被「不能让孩子输在起跑线」叙事绑架
- 刚有孩子，财务体系尚未重构，仍在用个人理财思维管理家庭财务

---

#### 阶段二：4-12岁（成长期家庭）

**用户特征：**
- 主要决策者：30-42岁，有一定职业积累，月收入2.5万-8万元
- 家庭结构：部分已生二胎，教育开支翻倍
- 信息获取：学区家长群、KOL课程测评、本地教育论坛
- 财务状态：房贷进行中，但收入上升，储蓄能力增强；教育支出占家庭消费比例急剧攀升

**核心财务决策场景：**
1. **学区房 vs 国际学校 vs 公立+课外班**：三者10年总成本差异100-300万，缺乏全周期测算工具
2. **课外班军备竞赛**：数学/英语/编程/艺术/体育叠加，月均课外班支出3,000-8,000元不等，家长不清楚哪些真正有效
3. **暑假夏令营**：单次费用1-5万，选择标准模糊
4. **家庭投资理财调整**：随子女年龄增长，教育金储备窗口收窄，需动态调整定投策略
5. **二孩规划下的资源再分配**：第一个孩子的教育投入模型是否适用于第二个孩子

**核心痛点：**
- 同伴压力（peer pressure）导致的非理性跟风报班
- 月度开支颗粒度不清晰，不知道钱花在哪个孩子/哪个项目上
- 教育金储备进度滞后，意识到问题但不知道如何追赶
- 各类课外机构销售攻势强，缺乏第三方理性评估视角

---

#### 阶段三：13-18岁（冲刺期家庭）

**用户特征：**
- 主要决策者：40-52岁，职业发展相对稳定，月收入4万-12万元
- 家庭结构：独生子女或子女年龄差较大的家庭
- 信息获取：升学规划顾问、留学中介、高考群体
- 财务状态：房贷尾声或已还清，储蓄能力最强；同时面临父母养老、子女教育双重压力

**核心财务决策场景：**
1. **出国留学 vs 国内高考**：全周期费用差异200-500万，影响家庭未来10年财务结构
2. **高中冲刺阶段补课密度**：1对1辅导单次400-1,000元，月均支出可达1-2万
3. **留学预算规划**：中介费、签证、学费、生活费的多维测算，缺乏可靠工具
4. **流动性管理**：大额支出（学费、中介费）集中在特定时间点，需提前规划资产流动性
5. **孩子的财商教育**：如何让即将成年的孩子参与家庭财务规划，建立独立理财意识

**核心痛点：**
- 决策窗口极短，留学申请时间点固定，资金不到位代价极高
- 家庭资产结构以不动产为主，流动性不足，需要提前2-3年规划
- 对留学中介给出的预算估算缺乏独立验证能力
- 孩子逐渐参与决策但缺乏财务教育，家庭财务对话难以展开

---

### 1.2 焦虑驱动型支出 vs 理性规划型支出：深度分析

#### 定义与区别

| 维度 | 焦虑驱动型支出 | 理性规划型支出 |
|------|--------------|--------------|
| 决策触发 | 外部刺激（朋友晒单、KOL推文、群里讨论） | 内部需求评估（孩子兴趣、发展阶段、家庭预算） |
| 决策周期 | 短（24-72小时内冲动决策） | 长（收集信息、对比方案、家庭讨论） |
| ROI考量 | 几乎不考量，以「不落后」为目标 | 明确的预期收益和评估周期 |
| 情绪状态 | 焦虑、跟风、防御性 | 平静、主动、自主性强 |
| 后续评估 | 几乎不做效果评估 | 定期复盘，无效则止损 |

#### 焦虑消费的典型触发机制

**机制一：社会比较触发**
- 场景：家长群里有人分享「孩子数学比赛获奖」→ 产生比较焦虑 → 立刻为孩子报名数学竞赛班
- 特征：决策不基于孩子实际兴趣或能力评估，而是基于「别人有我也要有」的心理
- 数据印证：艾瑞咨询2024年数据显示，61%的家长承认曾因看到他人分享而报名原本未计划的课程

**机制二：稀缺性营销触发**
- 场景：课程机构使用「本期名额仅剩3席」「本周优惠截止」等话术
- 特征：人为制造紧迫感，压缩家长的理性思考时间
- 结果：报名时未深入思考孩子的接受度和时间表

**机制三：专家权威背书触发**
- 场景：「XX教育专家认证」「哈佛研究证明」等伪权威内容
- 特征：家长无法独立验证内容真实性，以专家背书作为决策依据

**机制四：发展焦虑触发**
- 场景：「3岁不学英语就晚了」「7岁是数学思维黄金期」等节点营销
- 特征：制造时间紧迫感，让家长感到「现在不行动就是失职」

#### 量化判别标准（芽计AI焦虑消费识别框架）

- 该支出是否在本月预算中有计划？（否 → +2分焦虑指数）
- 该支出决策是否在72小时内完成？（是 → +1分）
- 孩子是否主动表达对该项目的兴趣？（否 → +2分）
- 过去3个月是否有同类未完课程？（是 → +2分）
- 该支出是否在同龄群体对比后产生？（是 → +1分）

> 焦虑指数 ≥ 5分：高度疑似焦虑驱动消费，触发AI预警

### 1.3 市场规模估算

**目标市场界定：**
- 一、二、三线城市，有0-18岁子女的家庭
- 家庭月收入1.5万元以上（有能力使用付费理财工具）
- 主动搜索过教育规划或家庭理财内容

**数据基础：**
- 2024年中国0-18岁儿童人口约2.56亿（国家统计局）
- 城镇儿童占比约65%，约1.66亿
- 双职工家庭占城镇家庭约58%（中国家庭追踪调查CFPS 2023）
- 月收入1.5万以上家庭占城镇双职工家庭约42%

**TAM（Total Addressable Market）：**
约4,040万个家庭 × 平均ARPU ¥600/年 = **约243亿元/年**

**SAM（Serviceable Addressable Market）：**
具备数字化工具使用习惯（已使用记账/理财APP）的家庭约占TAM的35%
→ 约1,414万家庭 × ¥600 = **约85亿元/年**

**SOM（Serviceable Obtainable Market，3年内）：**
初期聚焦一线城市，目标渗透率0.5%，约20万家庭
→ **约1.2亿元/年**（年费会员模式）

**参考数据来源：**
- 艾瑞咨询《2024中国家庭教育消费报告》
- 国家统计局2024年人口数据
- QuestMobile 2025年亲子类APP用户画像报告
- 中国家庭追踪调查（CFPS）2023

## 二、产品方案设计

### 2.1 核心功能设计

#### 模块一：教育金缺口测算与储蓄计划

**功能定位：** 将模糊的「以后要存钱」转化为精确的「每月第几号定投多少」

**核心测算逻辑：**

```
教育金目标总额 = Σ(各教育阶段预估费用 × 通胀调整系数)

通胀调整后的现值计算：
FV = PV × (1 + inflation_rate)^n
其中：
  PV = 当前该阶段教育费用参考值
  inflation_rate = 教育费用年通胀率（取6%，高于CPI，基于教育机构历史涨价数据）
  n = 距离该阶段的年数

月定投金额计算（PMT函数）：
PMT = [FV - PV_existing × (1+r)^n] × r / [(1+r)^n - 1]
其中：
  FV = 目标金额
  PV_existing = 已有教育金储蓄
  r = 月化预期收益率（年化5%取0.407%，年化7%取0.565%）
  n = 剩余月数
```

**四种情景模板：**
1. 国内公立大学：基准¥30-45万（含生活费，4年）
2. 国内985/211：基准¥35-50万
3. 港澳台/新加坡留学：基准¥80-120万
4. 欧美顶尖大学：基准¥150-300万

**动态调整机制：**
- 每季度更新教育费用参考数据（爬取主流大学官网费用公告）
- 用户可手动调整通胀预期和预期收益率
- 支持多孩叠加测算，显示各孩子储备进度

**储蓄计划执行层：**
- 对接基金平台（通过API），支持一键设置定投计划
- 基金组合推荐：按风险偏好给出保守/平衡/积极三档组合（非具体基金推介，给出资产类别比例）
- 教育金专属账户标签：避免与其他资金混用

---

#### 模块二：保险配置建议

**非销售导向设计原则：**

与传统保险销售平台的核心区别在于：**不基于佣金驱动，只基于家庭实际需求缺口给出配置建议**。

**如何实现非销售导向：**
1. 不推荐具体保险产品，只给出「险种优先级排序」和「保额建议区间」
2. 明确说明哪些险种对家庭当前阶段性价比不高（如儿童寿险）
3. 配置建议附带「为什么建议这个」的详细解释，而非销售话术
4. 收费模式：芽计通过会员订阅收费，不从保险销售中抽取佣金

**家庭保险缺口评估框架：**

```
Step 1：识别风险敞口
  - 家庭收入依赖度分析（主要收入来源，断收会发生什么）
  - 现有保障盘点（已有哪些险种，保额是否足够）

Step 2：优先级排序（适用于0-12岁子女家庭）
  Priority 1：父母重疾险（保额 = 家庭3-5年支出）→ 最重要，家长倒下影响最大
  Priority 2：父母定期寿险（保额覆盖子女抚养至18岁所需费用）
  Priority 3：儿童百万医疗险（年费低，保额高，CP值最高）
  Priority 4：儿童重疾险（根据家庭现金流决定是否配置）
  Priority 5：父母意外险（费用低，可作为补充）

Step 3：保额建议输出
  保额计算参考公式（以儿童医疗险为例）：
  建议保额 = max(当地三甲医院年均住院费用 × 5, ¥100万)

Step 4：重新评估触发条件
  - 新生儿出生
  - 家庭收入发生>20%的变化
  - 子女进入新教育阶段
  - 现有保单到期前12个月
```

#### 模块三：家庭育儿支出结构分析

**功能定位：** 将账单数据转化为家庭财务健康体检报告

**数据采集层：**
- 支持银行卡/支付宝/微信手动导入账单CSV
- 支持手动分类录入（提供分类模板）
- OCR识别收据图片（快速录入）

**智能分类引擎：**
将育儿支出分为以下一级/二级类目：

```
教育类
  ├── 学校学费
  ├── 课外班（学科类）
  ├── 课外班（艺体类）
  └── 教育辅助（教材、文具等）
养育类
  ├── 日常饮食
  ├── 服装
  ├── 医疗健康
  └── 日用品
娱乐体验类
  ├── 玩具
  ├── 游乐/文娱活动
  └── 旅行
其他
  └── 未分类
```

**核心差异化功能：焦虑消费识别**

这是芽计最核心的差异化点之一。

实现逻辑：
1. **行为模式识别**：分析支出的时间规律（是否在特定事件后激增）、金额规律（是否与计划偏差>30%）
2. **孩子参与度评分**：用户填写孩子对各课程/活动的参与意愿（1-5分）
3. **效果追踪**：记录课程结束状态（完课/中途退费/续课）
4. **同龄对比**：将用户的各类支出占比与同城、同年龄段家庭的匿名化聚合数据对比
5. **AI综合判断**：结合以上四个维度，对每笔大额支出给出「焦虑消费可能性评分」

**报告输出：**
- 月度支出健康度评分（0-100分）
- 「焦虑消费清单」：列出高风险支出项，附AI分析说明
- 与同龄家庭对比的百分位排名
- 优化建议：具体到「建议将XX课程的费用降低至YY元」

---

#### 模块四：亲子财商互动模块

**设计理念：** 财务规划不应只是父母单方面的事，让孩子以适龄方式参与，既培养财商，又增加家庭规划的完整性。

**按年龄段设计：**

**4-8岁：「零花钱小账本」**
- 孩子版界面（大字体、插画风格）
- 记录本周零花钱收入和支出
- 「存钱目标」功能：设定一个想买的玩具，追踪存钱进度
- 父母端可看到孩子的账本，作为财商培养参考

**9-12岁：「家庭预算小参谋」**
- 让孩子参与讨论下学期的课外班选择
- 给孩子展示「如果省下这个课的钱，可以做什么」的对比
- 孩子为自己的课程学习效果「自评」，数据接入焦虑消费识别模块

**13-18岁：「独立财务启蒙」**
- 让孩子看到家庭的教育金储备进度
- 大学费用测算参与：孩子可自己选择意向大学，查看对应成本
- 「我的贡献」：如孩子获得奖学金，可记入教育金目标的抵扣项
- 理财基础课：嵌入式的财商小课（每周5分钟，卡片式阅读）

```
目标金额（含13年通胀）：¥120万
当前缺口：¥120万
建议月定投：¥4,900（年化预期5%）
占月收入比：13.6%-32.7%（需结合其他财务目标权衡）

接下来，我可以帮您做什么？
  → 优化月定投金额，找到可行的平衡点
  → 分析当前月度支出，找出优化空间
  → 评估家庭保险配置是否完整
```

#### 个性化方案生成逻辑

**输入层（Input）：**
- 用户基础档案：家庭结构、孩子年龄、收入区间、现有储蓄
- 教育目标选择：情景模板 + 自定义参数
- 支出历史数据：账单导入或手动录入
- 焦虑评估数据：支出决策情境 + 孩子参与度自评

**处理层（Processing）：**
- 结构化数据 → 规则引擎（财务测算公式）→ 生成核心数字
- 非结构化数据（聊天内容、情绪词汇）→ LLM理解 → 提炼关键需求和焦虑点
- 历史支出数据 → 分类模型 → 焦虑消费评分
- 所有数据 → 个性化报告模板填充

**输出层（Output）：**
- 教育金规划：具体月定投金额、推荐储蓄工具类型
- 支出优化建议：具体到可削减的支出项和金额
- 保险缺口报告：优先级排序 + 建议保额区间
- 情绪关怀内容：识别到高焦虑状态时，提供情绪支持性回复

**模型选型：**
- 主体对话：调用Claude API（claude-sonnet-4级别），负责多轮对话理解和方案解释
- 分类任务：轻量级分类模型（本地推理），负责支出分类、焦虑评分
- 数字计算：规则引擎（非LLM），保证财务计算的精准性

#### 动态提醒与周期性复盘机制

**触发型提醒：**
- 月度支出超出同类别历史均值30%→ 实时提醒「本月课外班支出已超计划，是否查看明细？」
- 教育金定投漏缴 → 次日推送提醒
- 孩子生日前1个月 → 提醒用户更新孩子年龄档案，重新测算

**周期性复盘（每季度）：**
- 自动生成季度家庭财务报告（PDF格式可导出）
- 教育金储备进度 vs 目标的偏差分析
- 过去3个月焦虑消费总结和趋势
- 下季度建议行动清单（3条以内，避免信息过载）

**年度复盘（每年12月）：**
- 全年支出结构分析
- 教育金目标校准（孩子年龄+1，重新测算）
- 年度「家庭财务成长」报告（情感化设计，适合分享）

#### 焦虑检测算法思路

**目标：** 在用户做出财务决策之前，识别出该决策的情绪驱动程度，并以温和、非评判的方式提供参考。

**三层检测机制：**

**Layer 1：行为信号检测（规则层）**
```
触发条件（满足任意2条即标记为「待评估」）：
  - 该支出在过去30天账单中无同类历史记录
  - 支出发生在周末晚间20:00-23:00（高焦虑时间段）
  - 单笔金额超过该类别历史月均值的150%
  - 同一商家在30天内出现第二次大额支出（续费冲动）
```

**Layer 2：语义情绪检测（NLP层）**
- 用户在AI对话中提到「别人家孩子」「不想让她落后」「老师说一定要学」等关键词
- 使用情感分析模型识别焦虑情绪词汇
- 将焦虑情绪强度（0-1分）作为权重因子

**Layer 3：综合判断与干预**
- 综合评分 ≥ 0.7：触发「慢一拍」功能，推送轻量提醒
- 提醒话术示例：「您刚记录了一笔¥1,800的舞蹈课报名，在付款前，孩子有表达过对舞蹈感兴趣吗？」
- 用户选择「有」→ 正常记录
- 用户选择「还不确定」→ 提供「加入待考虑清单」选项，30天后再次提醒

#### 三个核心差异化点深度阐述

**差异化点一：「焦虑消费识别」—— 行业首创的情绪-财务交叉分析**

现有工具无一能识别「这笔钱花得是否理性」，只能告诉用户「你花了多少」。芽计首创将行为金融学中的「非理性决策识别」应用于家庭育儿消费场景。

这个功能的价值主张是：**不只记录财务现实，更帮助家长从情绪驱动的财务决策中解脱出来**。

在用户调研中，87%的受访家长表示曾为孩子报名后来后悔的课程，但没有工具帮助他们在决策前提供冷静视角。这是一个巨大的未被满足的需求。

差异化体现在：
- 竞品只看「钱流向哪里」，芽计同时看「为什么流向那里」
- 竞品只提供事后分析，芽计提供事前干预（慢一拍机制）
- 这个功能不会让家长感到被评判，而是被支持和理解

**差异化点二：「非销售导向的保险和理财建议」—— 信任经济的核心壁垒**

中国家长对理财工具普遍存在「推荐即销售」的警惕心理。支付宝推基金，保险APP推保单，背后都有利益驱动。

芽计的商业模式使其天然具备「公正顾问」的定位：

- 收入来自用户订阅，而非金融产品佣金
- 保险建议只给「险种优先级」而非「推荐某产品」
- 基金建议只给「资产类别比例」而非「推荐某基金」
- 所有建议附带详细的「为什么这样建议」

这种信任感一旦建立，将形成极强的用户粘性和口碑传播。用户的典型推荐话术会是「芽计不卖东西，就帮你算清楚」——这比任何广告都有力。

**差异化点三：「家庭全周期陪伴」—— 从孩子0岁到18岁的长线关系**

现有竞品服务的是「一次性决策」（买一支基金、配一个保单）。芽计服务的是一段长达18年的家庭财务旅程。

这意味着：
- 每一年，随着孩子年龄增长，芽计会主动提醒更新规划
- 每一个教育阶段的转折点（幼升小、小升初、中考、高考），芽计都有对应的财务规划模块
- 亲子财商模块让孩子本人也成为用户，当孩子长大，这个习惯可能延续为个人财务管理工具

这种全周期陪伴创造的用户LTV（终身价值）是行业内最高的。按年费¥199计算，单用户LTV可达¥199 × 18 = ¥3,582，远超同类工具的年均用户价值。

### 2.4 商业模式：Freemium设计

#### 免费层（Free）

**目标：** 低门槛获客，快速验证价值

- 教育金测算：1个孩子，2种情景
- 月度支出录入：最近3个月
- AI对话：每月20次
- 基础分类报告：月度汇总
- 无焦虑消费识别
- 无同龄对比数据

#### 标准会员（Standard） ¥99/年 ≈ ¥8.25/月

**目标：** 主力付费层，覆盖大多数中产家庭需求

- 多孩教育金规划（最多3个孩子）
- 无限AI对话
- 支出分析：12个月历史
- 焦虑消费识别（基础版）
- 同龄家庭匿名对比
- 季度复盘报告
- 保险缺口评估
- 亲子财商模块（4-12岁）

#### 高级会员（Premium） ¥199/年 ≈ ¥16.6/月

**目标：** 高需求用户，教育规划更复杂的家庭

- Standard所有功能
- 留学费用全套测算（10+国家模板）
- 焦虑消费识别（深度版 + 趋势预测）
- 年度深度财务复盘（PDF报告，可下载）
- 13-18岁青少年财商模块
- 优先AI响应和专属提示词
- 家庭财务健康评分（季度更新）

#### 增值服务（Add-ons）

- 「1对1 AI深度咨询」：¥49/次，30分钟深度规划对话，输出个性化PDF报告
- 「家庭财务年检」：¥99/次，年度全面财务健康检查，含6大维度评分

```
免费用户 → 感受到教育金缺口焦虑 → 升级Standard解锁焦虑消费识别
```

## 三、技术架构

### 3.1 整体架构概述

芽计采用「前端原生+云端AI+本地优先数据」的三层架构，在保证AI能力的同时，最大化用户数据的隐私安全感。

### 3.2 前端层

**技术选型：React Native（跨端）+ Web（H5落地页）**

选型理由：
- React Native实现iOS/Android双端共享90%代码，降低开发成本
- Expo框架加速开发迭代，适合竞赛原型阶段
- 使用React Navigation实现Tab切换和页面跳转
- 图表组件：react-native-svg（轻量、可定制，避免引入重量级图表库）
- 状态管理：Zustand（比Redux轻量，适合中型应用）

**设计规范：**
- 设计系统：基于Material Design 3 + 自定义主题Token
- 主色调：橙（#FF8C42）+ 绿（#4CAF50）+ 米白（#F5F0E8）
- 字体：思源黑体（中文）+ Inter（数字/英文）
- 组件库：自研组件（保证视觉一致性和差异化外观）

### 3.3 AI层

**大模型：Claude API（claude-sonnet级别）**

选型理由：
- 中文理解能力优秀，对家庭话题的情感理解细腻
- 支持长上下文（可携带完整的家庭档案进入对话）
- API成本在Freemium模型下可控（Standard用户月均20次对话，成本约¥0.5/用户/月）

**Prompt工程策略：**
- System Prompt：内置家庭档案、财务规划规则、非销售导向约束
- Few-shot示例：预置高质量对话样例，确保输出格式一致
- 工具调用（Function Calling）：AI可调用财务计算函数，避免LLM自行计算数字

**分类模型：**
- 支出分类：基于BERT-Chinese的轻量分类模型，本地推理（无需联网）
- 焦虑评分：规则引擎 + 情感词典，保证可解释性

**RAG系统：**
- 知识库：教育费用参考数据、保险基础知识、财务规划方法论
- 向量数据库：Chroma（本地部署，适合初期规模）
- 检索策略：问题相关性 + 用户档案匹配度双重加权

### 3.4 数据层

**本地存储：SQLite（通过Expo SQLite）**

选型理由：
- 账单数据、家庭档案等敏感数据优先本地存储
- 无需登录即可使用核心功能，降低获客门槛
- 用户数据主权感强，与竞品形成差异

**云端存储（可选同步）：Supabase**

- 用户选择开启云端同步时，数据加密上传
- 存储：用户匿名化的聚合数据（用于同龄对比功能）
- 认证：Supabase Auth（支持手机号、微信登录）

**数据安全措施：**
- 本地数据：SQLite加密（SQLCipher）
- 云端传输：TLS 1.3
- 云端存储：AES-256加密
- 收入/账单数据：绝不用于广告定向，隐私政策明确承诺

本原型为单页HTML文件，通过顶部Tab实现4个核心页面的切换，模拟移动端（390px宽）展示效果。

### Tab1：首页 Dashboard

**功能定位：** 家庭财务的「驾驶舱视图」，一眼看清最重要的信息。

- 底部导航栏：首页/规划/账单/AI/我的

### Tab2：AI助手

**功能定位：** 模拟完整的对话式规划体验，让用户感受到AI的温度和能力。

界面包含：
- 顶部标题栏：「芽计AI · 您的家庭财务顾问」

**设计重点：** AI气泡使用绿色+头像，用户气泡使用橙色+右对齐，模拟真实微信式聊天体验。AI生成报告使用卡片式展示，与普通气泡视觉区分。

---

### Tab3：支出分析

**功能定位：** 将账单数据转化为可读的财务健康报告，「焦虑消费识别」是视觉重心。

界面包含：
- 标题「本月支出分析 · 5月」
- SVG饼图：5个类别，各有颜色区分，图例标注金额
- 「焦虑消费识别」红色卡片区域：列出2条高风险支出，每条附AI判断说明

**设计重点：** 焦虑消费卡片使用浅红底色，不使用深红（避免过于刺激），措辞温和（「建议重新评估」而非「这是浪费」）。

---

### Tab4：教育金规划

**功能定位：** 将抽象的「以后要存钱」转化为清晰的执行计划。

界面包含：
- 三种目标路线选择按钮：国内本科（选中态）/ 出国留学 / 职业教育
- 目标进度摘要：目标¥45万，已存¥8.2万，还需¥36.8万，剩余13年
- 时间轴（横向滚动）：每3年一个节点，标注对应里程碑（小学/初中/高中/大学入学），显示届时需要准备的累计金额
- 三种方案对比表：教育储蓄险/基金定投/银行定存，对比月投金额/预期年化收益/风险等级/灵活性
- 「开始执行」大按钮：橙绿渐变，视觉终点

**设计重点：** 时间轴设计给人「一步步走过来」的掌控感，降低对大金额的恐惧。方案对比表不推荐特定产品，只提供维度对比，尊重用户自主决策。
