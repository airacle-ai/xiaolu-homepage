📖 这是什么

这是一套围绕 AI 短剧 制作的完整工具库，覆盖从「故事怎么选 → 怎么写 → 怎么发声」的全流程，分两个独立但互补的子模块。

第一个子模块 · 选题与脚本库：基于 2026 年抖音/红果/AI 漫剧最新爆款数据，整理了 58 个完整选题（含选题原因+故事梗概+爆款打分），并写了 6 个 S 级题材的完整第一集脚本，外加伪纪录片 × 伪综艺两个新形态的完整策划方案。

第二个子模块 · 配音音色库：把脚本里的角色直接对应到火山引擎豆包 TTS 大模型的 7 个真实音色 ID，提供 Python 客户端可一行命令生成配音，已封装好声音物理参数（音调/语速/情感/气声等）。

⚡一行命令生成配音

# 用角色名调用（自动套用预设参数） $ python tts_client.py --character 苏棠 $ python tts_client.py --character 林晚 --text "奶奶，我找到了。" # 直接用音色 ID $ python tts_client.py --voice S_uxrEEuK42 --text "哎哟喂！" # 批量为所有角色生成样例（7 个音频文件） $ python tts_client.py --batch # 列出所有可用音色 $ python tts_client.py --list
